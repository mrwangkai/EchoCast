### Objective
Convert Browse from a standalone tab to a pushed navigation page, reducing tab bar clutter and using search icon in Home header as the primary ingress point.

### Changes Made

#### 1. ContentView.swift ŌĆö Remove Browse Tab
- Removed `PodcastDiscoveryView` tab (was tag 1)
- Updated `LibraryView` tag: 2 ŌåÆ 1
- Removed `selectedTab` binding from `HomeView()` and `LibraryView()`

**Before:**
```swift

    HomeView(selectedTab: $selectedTab).tag(0)
```

**After:**

#### 2. HomeView.swift ŌĆö Add NavigationPath
- Added `@State private var navigationPath = NavigationPath()`
- Updated `NavigationStack` to use path: `NavigationStack(path: $navigationPath)`
- Added `.navigationDestination(for: String.self)` for "browse" ŌåÆ `PodcastDiscoveryView()`
- Replaced three `selectedTab = 1` calls with `navigationPath.append("browse")`:
  - Search icon button (header)
  - "Find more" button (Following section)
  - Empty state "Find a podcast" button
- Removed unused `@Binding var selectedTab: Int`

**Key Implementation:**
```swift

    ScrollView { ... }
        .navigationDestination(for: String.self) { destination in
            if destination == "browse" {
                PodcastDiscoveryView()
            }
        }
}
```

#### 3. LibraryView.swift ŌĆö Add NavigationPath
- Same pattern as HomeView
- Added `@State private var navigationPath = NavigationPath()`

#### 4. PodcastDiscoveryView.swift ŌĆö Remove Inner NavigationStack
**Root Cause**: PodcastDiscoveryView had its own `NavigationStack` which created a nested NavigationStack when pushed via navigation from HomeView, causing blank screen rendering.

**Fix:** Removed the inner `NavigationStack` wrapper ŌĆö PodcastDiscoveryView now uses the parent NavigationStack's navigation context when pushed.

**Before:**
```swift

    NavigationStack {  // ŌåÉ Removed
        ScrollView { ... }
    }
}
```

**After:**
```swift

    ScrollView { ... }
}
```

#### 5. Polish ŌĆö Fix Header Flash

### Issues Encountered & Resolved

#### Issue 1: Blank Screen on Navigation Push
**Symptom:** Tapping search icon showed blank white screen with back caret

**Diagnosis:** `.navigationDestination` modifier was placed after view transform modifiers (`.background`, `.navigationBarHidden`, `.onAppear`)

**Fix:** Moved `.navigationDestination` to be the first modifier on ScrollView

**Commit:** `72d973c`

#### Issue 2: Nested NavigationStack Conflict
**Symptom:** Blank screen with warning triangle after initial fix

**Diagnosis:** PodcastDiscoveryView contained its own `NavigationStack` at line 27, creating nested NavigationStacks when pushed via navigation

**Fix:** Removed inner NavigationStack wrapper from PodcastDiscoveryView

**Commit:** `30bde23`

#### Issue 3: Header Visual Flash
**Symptom:** Navigation title flashed when pushing to Browse

**Fix:** Changed `.navigationBarTitleDisplayMode(.large)` to `.inline`

**Commit:** `4519612`
